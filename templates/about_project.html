Problem Statement

The project aims to create a system that can recognize human facial expressions in real-time and provide analytics on the emotional state of individuals over a session. This can be used in various fields like mental health, education, and marketing.

Solution Overview

We have developed a web-based application that captures video from a webcam, detects faces, and recognizes the facial expressions using a pre-trained deep learning model. The application then stores this data and provides a dashboard for analytics.

Tech Stack

  • Python
  • OpenCV
  • PySpark
  • ANN
  • Firebase
  • Firestore
  • TensorFlow

Use Cases

  • Mental health monitoring
  • Interview analysis
  • Student engagement
  • Workplace productivity
