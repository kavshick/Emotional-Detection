Dataset Used

The model was trained on the FER-2013 dataset, which contains 35,887 grayscale images of faces, each 48x48 pixels. The faces are categorized into seven emotions: angry, disgust, fear, happy, sad, surprise, and neutral.

Training Method

The model is an Artificial Neural Network (ANN) built and trained using PySpark. PySpark's distributed computing capabilities allow for efficient training on large datasets.

Cross-Validation and Accuracy

We used k-fold cross-validation to ensure the model's robustness and prevent overfitting. The final model achieved an accuracy of over 90% on the test set.
